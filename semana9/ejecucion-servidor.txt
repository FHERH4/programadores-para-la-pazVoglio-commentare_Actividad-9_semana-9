1. ¿Qué ocurre cuando ejecutas node server.js?
El proceso se queda ejecutándose en la terminal de forma continua, bloqueando la línea de comandos para mantenerse activo y escuchando las peticiones entrantes.

2. ¿Qué mensaje aparece en la terminal?
Servidor corriendo en el puerto 3000

3. ¿Qué sucede cuando visitas la ruta /estado?
El navegador web se conecta exitosamente al servidor a través del puerto reenviado de GitHub Codespaces y muestra un objeto en formato JSON con los datos del estado del servidor y el nombre de la API comunitaria.

{
  "estado": "Servidor funcionando",
  "servicio": "API comunitaria"
}
